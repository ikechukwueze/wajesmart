Question 1

A Q() object represents an SQL condition that can be used in database-related operations.

For example, consider the model below

from django.db import models

class Person(models.Model):
 first_name = models.CharField(max_length=30)
 last_name = models.CharField(max_length=30)
 age = models.IntegerField()

To filter records with first name of "John" OR an age > 18, the corresponding query would be:

query = Person.objects.filter(Q(first_name='John') | Q(age__gt=18))
